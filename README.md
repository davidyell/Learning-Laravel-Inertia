# Learning Laravel Inertia

A test project to learn some Laravel with Intertia using Vue.js

It was setup using the Breeze starter kit, https://laravel.com/docs/11.x/starter-kits#breeze-and-inertia
Plus it makes use of https://flowbite-vue.com/

## Install

-   Clone the repo
-   `composer install`
-   `./vendor/bin/sail up -d`
-   `./vendor/bin/sail artisan migrate:fresh --seed`

Visit http://localhost:8080
